# Calculator Project

Java-аар бичигдсэн энгийн calculator төсөл.

## Хэрхэн ажиллуулах

### Тест ажиллуулах:
```bash
mvn test
```

### Checkstyle шалгах:
```bash
mvn checkstyle:check
```

### Code Coverage тайлан үүсгэх:
```bash
mvn jacoco:report
```
Тайлан: `target/site/jacoco/index.html`

## Шаардлага
- Java 17
- Maven 3.6+
